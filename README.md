# 🛒 Django E-Commerce Web Application

This is a **full-stack E-Commerce web application** built using **Django**.
The application allows users to register, log in, browse products by category, add products to a cart, and place orders.

---

# 🚀 Features

* User Registration and Login system
* Browse all products
* View product details
* Product categories filtering
* Add products to cart
* Remove items from cart
* Order placement system
* Django Admin panel for product management

---

# 🛠 Tech Stack

* **Backend:** Django (Python)
* **Database:** SQLite
* **Frontend:** HTML, CSS
* **Authentication:** Django Authentication System

# ⚙️ Installation & Setup

### 1 Clone repository

```
git clone https://github.com/yuvasri200/E-commerce-Website-.git
```

### 2 Navigate to project

```
cd django-ecommerce
```

### 3 Create virtual environment

```
python -m venv venv
```

### 4 Activate virtual environment

Windows:

```
venv\Scripts\activate
```

### 5 Install dependencies

```
pip install -r requirements.txt
```

### 6 Run migrations

```
python manage.py migrate
```

### 7 Run server

```
python manage.py runserver
```

Open browser:

```
http://127.0.0.1:8000
```

---

# 👨‍💻 Admin Panel

Create admin user:

```
python manage.py createsuperuser
```

Open admin panel:

```
http://127.0.0.1:8000/admin
```
